    Tokyo Trials Literature Database

    Alternate Title
    东京审判文献数据库
    Category
    Archival Material
    Access
    subscription
    URL
    http://www.tokyotrial.cn/
    Description
    Full-text searchable database of documents and audio-visual resources covering the trials of World War II Japanese war criminals. The collection is published in stages. All documents will eventually be available in English, Chinese and Japanese. To use the database please click on the button “Log In” button in the top menu and choose “Institutional User Login” without filling in anything, to proceed. Access Note If off-campus, please access using the SonicWall VPN .
